# 说明

This is a demo for multi language test.

Qt creator 语言专家支持必须安装sudo apt-get install qttools5-dev-tools.

使用lupdate生成多语言ts文件方法如下
Creator菜单栏，选择：工具->外部->Qt语言家->更新翻译（lupdate）
命令行：lupdate qt-multi-lan.pro
